## 一、项目定位

### 核心卖点

| 卖点 | 说明 |
|------|------|
| 一套代码，全端运行 | H5、微信小程序、App（iOS/Android）、HarmonyOS |
| 开箱即用 | 无需自建服务器，uniCloud 免费额度足够小团队使用 |
| Vue 技术栈 | Vue 3 + Vuex，降低学习和二次开发成本 |
| MIT 开源 | 可商用、可二次开发、无版权顾虑 |
| 中文友好 | 文档、代码注释全中文，国内开发者无障碍使用 |

### 目标用户

1. **uni-app 开发者**：寻找完整项目参考或快速搭建团队工具
2. **小型团队/工作室**：需要轻量级任务管理，不想付费或自建服务器
3. **学习者**：想学习 uni-app + uniCloud 全栈开发的开发者

## 七、文案模板

### 插件市场描述（草稿）

```
uni-task - 轻量级团队任务管理系统

基于 uni-app + uniCloud 开发，一套代码多端运行。

功能特性：
- 项目管理：创建、归档、成员管理、邀请码加入
- 任务管理：CRUD、优先级、截止日期、分组拖拽排序
- 子任务：支持任务分解
- 附件管理：文件上传与管理
- 操作日志：自动记录项目动态

技术栈：Vue 3 + Vuex + uniCloud + uni-id

开源协议：MIT（可商用）
```
